e:Te0c,<p>This is so exciting!!! and it is just the beginning...</p>
<p>So this is the first post and the presentation of a new game <strong>Arkapongout</strong>, if you don't need an explanation for what kind of game it I already like you and if you don't... well don't worry, it happens! the shortest way to describe it is</p>
<h3 id="arkanoid--pong--breakout"><a aria-hidden="true" tabindex="-1" href="#arkanoid--pong--breakout"><span class="icon icon-link"></span></a><a href="https://en.wikipedia.org/wiki/Arkanoid" rel="nofollow">Arkanoid</a> + <a href="https://en.wikipedia.org/wiki/Pong" rel="nofollow">Pong</a> + <a href="https://en.wikipedia.org/wiki/Breakout" rel="nofollow">Breakout</a></h3>
<p>Yup, pretty much that three words (And links to their definition if you still not knowing them) say everything about the game and its mechanics.</p>
<img src="/posts/assets/ghost-posts-images/2017/05/arkapongout-00.png" alt="Arkapongout Prototype 1 Version">
<p>Up there you can see a picture with the current stage of the game, as you can see it is in a very simple <strong>prototype</strong> with no so mutch mechanics, graphics or features, let's be honest, there's is nothing innovative yet in the game but it's playable.</p>
<p>But don't be sad about that simplicity, It will receive much more improvements, features and new mechanics.</p>
<h3 id="play-it"><a aria-hidden="true" tabindex="-1" href="#play-it"><span class="icon icon-link"></span></a><a href="https://zombie-unicorn.itch.io/arkapongout" rel="nofollow">Play It!</a></h3>
<p>Yes!!! You read that right, <strong>you can play it now</strong> and rage over it on the comments (but please, be kind).</p>
<p>Jokes apart, feel free to <strong>request any improvements as well as report bugs</strong>. You can reach me by the <a href="https://zombie-unicorn.itch.io/arkapongout" rel="nofollow">itch.io comments</a>, by submitting issues in the <a href="https://github.com/ZombieUnicornStudio/Arkapongout/issues" rel="nofollow">Game GitHub project</a> or <a href="https://twitter.com/arkapongout" rel="nofollow">@Arkapongout on twitter</a>.</p>

<p>I'll try to <strong>publish</strong> one new version as well a little entry of what has changed in the game <strong>each 2 weeks or once a month</strong>, It'll depend on the amount of work and my free time.</p>
<p>The game is only being developed by <a href="http://onlythepixel.com" rel="nofollow"><strong>me</strong></a> but it could change over the time (I enjoy a lot working in team, so feel free to contact me if you want to collaborate). A con is that the speed of development will be slow and probably sometimes the changes will not be so big but let's have patience on this. This is because I'm <strong>developing the game in my free time</strong> so sometimes I'll not being able to spend all the time I want on it [picture my sad face in your mind].</p>
