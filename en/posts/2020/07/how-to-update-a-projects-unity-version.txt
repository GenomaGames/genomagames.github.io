d:Td4d,<p>Learn how to upgrade your project's Unity Version. We will update a Unity project using Unity 2020.1.0b13 (Beta) to Unity 2020.1.0f1.</p>
<!--more-->
<h2 id="requirements"><a aria-hidden="true" tabindex="-1" href="#requirements"><span class="icon icon-link"></span></a>Requirements</h2>
<p>To follow this Unity tutorial you will need to:</p>
<ul>
<li><a href="https://unity3d.com/es/get-unity/download" rel="nofollow">Install UnityHub</a></li>
<li><a href="./how-to-install-unity-2020.md">Install Unity</a></li>
</ul>
<h2 id="installing-a-new-unity-version"><a aria-hidden="true" tabindex="-1" href="#installing-a-new-unity-version"><span class="icon icon-link"></span></a>Installing a new Unity Version</h2>
<p>We already had a Unity version installed, <a href="https://blogs.unity3d.com/es/2020/03/17/unity-2020-1-beta-is-now-available-for-feedback/" rel="nofollow">Unity 2020.1.0b13 (Beta)</a>, and a new one has been released, <a href="https://blogs.unity3d.com/es/2020/07/23/unity-2020-1-is-now-available/" rel="nofollow">Unity 2020.1.0f1</a>.</p>
<p>First, you will need to install Unity's new version, if you used <strong>Unity Hub</strong> to install the previous one, the process is the same, you can follow the steps in <a href="./how-to-install-unity-2020"><strong>How to Install Unity 2020 Tutorial</strong></a>.</p>
<img src="/posts/assets/2020-07-24-how-to-update-a-projects-unity-version/01-Unity_Hub_Installs_with_new_version_installed.png" alt="01-Unity_Hub_Installs_with_new_version_installed">
<h2 id="updating-projects-unity-version"><a aria-hidden="true" tabindex="-1" href="#updating-projects-unity-version"><span class="icon icon-link"></span></a>Updating Project's Unity Version</h2>
<p>Before upgrading your project's Unity version, checkout Unity Manual's section <a href="https://docs.unity3d.com/2020.1/Documentation/Manual/UpgradeGuides.html" rel="nofollow"><strong>Upgrade Guides</strong></a>, this way you can be sure if your project is ready to be upgraded or if you need to make changes in it before attempting to upgrade it.</p>
<p>To update the Unity Version of a project, go to <strong>Projects</strong> section and click on the select input to choose the newly installed version.</p>
<img src="/posts/assets/2020-07-24-how-to-update-a-projects-unity-version/02-Unity_Hub_Project_Unity_version_selection.png" alt="02-Unity_Hub_Project_Unity_version_selection">
<p>After selecting the new Unity version you will need to open the project, Unity Hub will show an alert dialog informing that the project's version is going to be updated and where Unity will place the logs of the upgrade process.</p>
<img src="/posts/assets/2020-07-24-how-to-update-a-projects-unity-version/03-Unity_Hub_Project_Unity_version_upgrade_alert.png" alt="03-Unity_Hub_Project_Unity_version_upgrade_alert">
<p>Press the <strong>CONFIRM</strong> button, and the new Unity editor will proceed to update the project's references and dependencies.</p>
<img src="/posts/assets/2020-07-24-how-to-update-a-projects-unity-version/04-Unity_Project_open_in_progress.png" alt="04-Unity_Project_open_in_progress">
<p>When Unity finishes, you will see the Unity editor with your project again, running in a new version.</p>
<img src="/posts/assets/2020-07-24-how-to-update-a-projects-unity-version/05-Unity_Project_opened.png" alt="05-Unity_Project_opened">
<p>We are done!</p>
