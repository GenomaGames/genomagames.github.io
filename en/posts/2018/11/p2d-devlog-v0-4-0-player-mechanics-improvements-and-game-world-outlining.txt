15:T107e,<p>Another step towards a better game! This one almost appears to be a step back, the created scenes have been removed but in exchange, there is a huge world to be shaped.</p>
<p>You can check the <a href="../09/p2d-devlog-v0-3-0-enemies-and-new-location">previous devlog</a> if you missed it.</p>
<h2 id="tldr"><a aria-hidden="true" tabindex="-1" href="#tldr"><span class="icon icon-link"></span></a>TL;DR;</h2>
<ul>
<li>Defined the outline of the game world waiting to be shaped</li>
<li>Improve player movements, now jumping, running and attacking feel more organic</li>
<li>The player can now attack while on air</li>
<li>Now the game loads the areas progressively without load screens</li>
<li>Implemented a pause menu</li>
<li>Now the game works on iPhone and Android and it will run at 60fps</li>
<li>The whole game logic has been improved (game management, inputs, menus, enemies...)</li>
</ul>
<blockquote>
<p>Don't miss the GIFs! Scroll a little bit</p>
</blockquote>
<h2 id="detail"><a aria-hidden="true" tabindex="-1" href="#detail"><span class="icon icon-link"></span></a>Detail</h2>
<p>The biggest change in this version is <strong>the world</strong> and the <strong>level progression mechanic</strong> followed by improvements in several game mechanics and systems (a better core a better game)</p>
<h3 id="the-world"><a aria-hidden="true" tabindex="-1" href="#the-world"><span class="icon icon-link"></span></a>The World</h3>
<p>The whole world outline has been designed, meaning that now the game can be played from beginning to end, there's no content yet so the player walks along plain and empty landscapes ready to be shaped with more interesting forms and to be filled with enemies, objects, decoration...</p>
<img src="https://i.imgur.com/sBa4xAd.gif" alt="Whole World Outline">
<p>You can complete the game as it is now in around 15 minutes. Now the challenge is to fill and expand that time width fun and challenging scenery.</p>
<h3 id="the-new-level-loading-system"><a aria-hidden="true" tabindex="-1" href="#the-new-level-loading-system"><span class="icon icon-link"></span></a>The new level loading system</h3>
<p>The old screen fading to change between levels has been replaced for a progressive loading system. No more loading screens.</p>
<img src="https://i.imgur.com/lYso4er.gif" alt="New Level Loading System">
<p>Although it is likely that the screen fade system will be used to pass through doors.</p>
<h3 id="movement-improvements"><a aria-hidden="true" tabindex="-1" href="#movement-improvements"><span class="icon icon-link"></span></a>Movement Improvements</h3>
<p>The movement logic has been changed to be a little more organic. The player and other walking things now have a small acceleration time, this allows the player to make more precise movements.</p>
<img src="https://i.imgur.com/IDUkZBE.gif" alt="Player Movement Improves">
<p>It could be improved more but at this moment it's fine.</p>
<h3 id="air-attack"><a aria-hidden="true" tabindex="-1" href="#air-attack"><span class="icon icon-link"></span></a>Air Attack!</h3>
<p>In the previous version the game included a new type of enemy (bats), now the player can fight them by attacking while jumping.</p>
<img src="https://i.imgur.com/iKHQIBl.gif" alt="Player now can attack while on air">
<h3 id="menus"><a aria-hidden="true" tabindex="-1" href="#menus"><span class="icon icon-link"></span></a>Menus</h3>
<p>A new pause menu has been added to access to the main menu while on gameplay (or to take a break)</p>
<img src="https://i.imgur.com/5rpZXGp.gif" alt="Pause Menu">
<h2 id="nobody-thinks-in-the-future"><a aria-hidden="true" tabindex="-1" href="#nobody-thinks-in-the-future"><span class="icon icon-link"></span></a>Nobody thinks in the future?</h2>
<p>The plan now is:</p>
<ul>
<li>Forest zone levels content</li>
<li>Dungeon zone levels content</li>
<li>Work on the game world and leveling</li>
<li>More enemies</li>
<li>New mechanics</li>
</ul>
